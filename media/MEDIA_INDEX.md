# Media Index — CellAutomata generative art

All paths are under `media/` unless noted. Everything here is committed to the
branch `claude/cool-shannon-Sa9Tu` **except** the one huge 4K master noted below
(gitignored — delivered via chat only).

## How to get these onto your machine (e.g. F:\temp)
This repo is the transport. From your Windows machine:
```
cd F:\temp
git clone -b claude/cool-shannon-Sa9Tu https://github.com/rizzleroc/CellAutomata.git
```
→ all videos land in `F:\temp\CellAutomata\media\`. Or, in an existing checkout:
`git pull origin claude/cool-shannon-Sa9Tu`.

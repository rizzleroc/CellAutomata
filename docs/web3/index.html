W E L C O M E

cellauto — the building blocks of life

Fourteen live simulations, in canonical chemistry-to-life order. Each one is a separate building block — a different way the universe organises itself into something life-like. Each one is controlled by sliders the right-hand panel labels in plain English. New in v3.0 (per PRD v4.1): every rule renders as a depth-shaded SEM substrate with a bioform sprite layer composited on top, so what you see reads as a microscope image of the chemistry, not a heat-map. Press space to pause, 1–0 to jump the first ten rules, t to start a guided tour.
